#  Gestão de Clientes

Sistema completo de gestão de clientes com autenticação de usuários e operações CRUD, desenvolvido com foco em boas práticas e arquitetura full stack.

---

##  Funcionalidades

* Cadastro de usuários
* Login com autenticação
* Controle de acesso
* CRUD completo de clientes
* Integração com API

---

##  Tecnologias

* C# (.NET)
* JavaScript
* HTML & CSS
* Bootstrap
* SQL

---

##  Estrutura do Projeto

```
ApiClientes   → API em C#
ClientesFRONT → Interface web
```

---

##  Como executar

### Backend:

```
cd ApiClientes
dotnet run
```

### Frontend:

Abra o arquivo HTML no navegador

Projeto desenvolvido para fins de portfólio, demonstrando habilidades em desenvolvimento full stack, autenticação e integração com APIs.
