#  Gestão de Clientes

Sistema completo de gestão de clientes com autenticação de usuários e operações CRUD, desenvolvido com foco em boas práticas e arquitetura full stack.

---![gestaofuncionando](https://github.com/user-attachments/assets/63ad18a8-e72a-4fd5-83c7-b8e6a8739bec)

## 🧩 Sobre o sistema

O sistema permite que usuários se cadastrem e realizem login para acessar um painel administrativo, onde é possível gerenciar clientes através de operações CRUD. A aplicação utiliza uma API para comunicação entre frontend e backend, garantindo separação de responsabilidades.

##  Funcionalidades

* Cadastro de usuários
* Login com autenticação
* Controle de acesso
* CRUD completo de clientes
* Integração com API
